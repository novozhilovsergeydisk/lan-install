Отчет по изменениям перед деплоем:

1. Логика и изменения:
Внесены изменения в файл `public/robots.txt`. Добавлена директива `Disallow: /`, которая запрещает поисковым системам сканировать и индексировать содержимое всего сайта. 

2. Возможные регрессии:
Регрессий нет. Изменение затрагивает только поведение внешних поисковых роботов. Это никак не скажется на работоспособности веб-приложения для конечных пользователей или на взаимодействии с API Яндекс.Карт.

3. Безопасность и чувствительные данные:
Чувствительные данные (пароли, токены) отсутствуют в изменениях. Изменение само по себе повышает приватность проекта, скрывая его от публичного поиска.

Предлагаемый комментарий для коммита: "chore: запрещена индексация сайта в поисковых системах"
